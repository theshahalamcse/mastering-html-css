CSS Background

The CSS background property is a shorthand property that allows you to set multiple background-related properties in one declaration. It can include values for background-color, background-image, background-position, background-size, background-repeat, and background-attachment.

1. Background Color

You can set a background color using the background-color property:

background-color: lightblue;

2. Background Image

You can set a background image using the background-image property:

background-image: url('image.jpg');

3. Background Position

You can specify the position of the background image:

background-position: center;

4. Background Size

You can control the size of the background image:

background-size: cover;

5. Background Repeat

You can specify whether the background image should repeat:

background-repeat: no-repeat;

6. Background Attachment

You can control whether the background image scrolls with the page:

background-attachment: fixed;
