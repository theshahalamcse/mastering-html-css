HTML Text Formatting

HTML provides several tags to format text. These tags allow you to change the appearance of text, such as making it bold, italic, underlined, or adding emphasis. Here are some common HTML text formatting tags:

  • <b>: Makes text bold.
  • <i>: Makes text italic.
  • <u>: Underlines text.
  • <em>: Emphasizes text (usually italic).
  • <strong>: Strongly emphasizes text (usually bold).
  • <mark>: Highlights text.

Example:

<h2>This is a heading</h2>
<p>This is a <b>bold</b> word.</p>
<p>This is an <i>italic</i> word.</p>
<p>This is an <u>underlined</u> word.</p>
<p>This is an <em>emphasized</em> word.</p>
<p>This is a <strong>strongly emphasized</strong> word.</p>
<p>This is a <mark>highlighted</mark> word.</p>

When you use these tags, the text will be displayed with the specified formatting. For example, the word "bold" will appear in bold, "italic" will appear in italics, and so on. You can also combine these tags to apply multiple formatting styles to the same text.

Remember that these tags are used for presentation purposes, and it's important to use them appropriately to ensure that your content is accessible and semantically correct.

  • <b>: Makes text bold.
  • <i>: Makes text italic.
  • <u>: Underlines text.
  • <em>: Emphasizes text (usually italic).
  • <strong>: Strongly emphasizes text (usually bold).
  • <mark>: Highlights text.
  • <small>: Makes text smaller than the surrounding text.
  • <del>: Indicates deleted text (usually displayed with a strikethrough).
  • <ins>: Indicates inserted text (usually displayed with an underline).
  • <sub>: Makes text subscript (lower than the normal text).
  • <sup>: Makes text superscript (higher than the normal text).
