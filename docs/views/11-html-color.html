HTML Color

HTML color is used to specify the color of text, backgrounds, and other elements on a webpage. There are several ways to define colors in HTML.

Color Syntax

Colors can be specified using:

  • Color names: e.g., "red", "blue", "green"
  • Hex values: e.g., "#ff0000", "#00ff00", "#0000ff"
  • RGB values: e.g., "rgb(255, 0, 0)", "rgb(0, 255, 0)", "rgb(0, 0, 255)"
