HTML Images

In HTML, you can add images to your web pages using the <img> element. The src attribute specifies the path to the image file, and the alt attribute provides alternative text for accessibility.

Image Attributes

  • src: Specifies the URL of the image.
  • alt: Provides alternative text for screen readers.
  • width: Sets the width of the image (in pixels or percentage).
  • height: Sets the height of the image (in pixels or percentage).

You can also use CSS to style images further, such as adding borders, shadows, or adjusting their size and position on the page.
