HTML Table

Tables are used to display data in a tabular format (rows and columns).

Basic Table Structure

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Example Table

Name Age City
Alice 30 New York
Bob 25 Los Angeles
