CSS Border

The CSS border property is used to define the border of an HTML element. It is a shorthand property that allows you to set the width, style, and color of the border in a single declaration.

Border Syntax

The syntax for the border property is as follows:

border: [border-width] [border-style] [border-color];

Border Properties

  • border-width: Specifies the width of the border (e.g., 1px, 2em).
  • border-style: Specifies the style of the border (e.g., solid, dashed, dotted).
  • border-color: Specifies the color of the border (e.g., red, #ff0000).

Example

Here is an example of how to use the border property:

div { border: 2px solid blue; }

This will create a blue solid border with a width of 2 pixels around any <div> element.
