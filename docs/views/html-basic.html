Few HTML BASIC Concepts

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure and content of a webpage, allowing developers to define elements such as headings, paragraphs, links, images, and more. HTML uses tags to indicate different types of content and their relationships, making it an essential component of web development.

Example of HTML Structure

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Sample HTML Document</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
                <p>This is a sample HTML document.</p>
            </body>
            </html>

Heading Tags

HTML provides six heading tags, from <h1> to <h6>, which are used to define the hierarchy of headings on a webpage. The <h1> tag represents the most important heading, while <h6> represents the least important.

  • <h1> - Main heading
  • <h2> - Subheading
  • <h3> - Sub-subheading
  • <h4> - Sub-sub-subheading
  • <h5> - Sub-sub-sub-subheading
  • <h6> - Sub-sub-sub-sub-subheading

Paragraph Tags

The <p> tag is used to define a paragraph of text in HTML. It is a block-level element that creates a new line before and after the content it contains. Paragraphs are essential for organizing and structuring text on a webpage, making it easier for users to read and understand the content.

Example of a paragraph:

            <p>This is a paragraph of text in HTML. It provides information about the content and helps to structure the webpage.</p>

Link Tags

The <a> tag is used to create hyperlinks in HTML. It allows users to click on the link and navigate to another webpage or a different section of the same page. The <a> tag uses the "href" attribute to specify the destination URL.

Example of a link:

            <a href="https://www.example.com">Visit Example.com</a>

Image Tags

The <img> tag is used to embed images in an HTML document. It is a self-closing tag that requires the "src" attribute to specify the image source and the "alt" attribute to provide alternative text for accessibility.

Example of an image:

            <img src="image.jpg" alt="Description of the image">

List Tags

HTML provides two types of list tags: ordered lists (<ol>) and unordered lists (<ul>). Ordered lists are used for items that have a specific order, while unordered lists are used for items that do not have a specific order. Each item in the list is defined using the <li> tag.

Example of an ordered list:

            <ol>
                <li>First item</li>
                <li>Second item</li>
                <li>Third item</li>
            </ol>

Example of an unordered list:

            <ul>
                <li>First item</li>
                <li>Second item</li>
                <li>Third item</li>
            </ul>
