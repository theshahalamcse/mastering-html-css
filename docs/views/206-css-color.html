CSS Colors

In CSS, colors can be specified using various formats. Here are some common ways to define colors:

1. Named Colors

You can use predefined color names:

color: red;

2. Hexadecimal Colors

Colors can be defined using hexadecimal values:

color: #ff0000; /* Red */

3. RGB Colors

Colors can also be defined using the RGB format:

color: rgb(255, 0, 0); /* Red */

4. RGBA Colors

The RGBA format allows you to specify an alpha (opacity) value:

color: rgba(255, 0, 0, 0.5); /* Semi-transparent Red */

5. HSL Colors

You can define colors using the HSL format:

color: hsl(0, 100%, 50%); /* Red */
