HTML CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to control the color, font, spacing, and overall appearance of your HTML elements.

CSS Syntax

CSS rules consist of a selector and a declaration block:

selector {
    property: value;
    }

Example of CSS Styling

This text is styled with CSS.

This text has a yellow background.

In the example above, different styles are applied to paragraphs using the `style` attribute. You can also use external or internal CSS to apply styles more efficiently.

  • Inline CSS: Using the `style` attribute directly on an HTML element.
  • Internal CSS: Placing CSS rules within a <style> tag in the <head> section of the HTML document.
  • External CSS: Linking to an external CSS file using the <link> tag in the <head> section.
