HTML Comments

HTML comments are used to add notes or explanations within your HTML code that will not be displayed in the browser. They are helpful for documenting your code or temporarily hiding parts of it during development.

Comment Syntax

An HTML comment starts with <!-- and ends with -->. Anything between these markers will be treated as a comment and ignored by the browser.

Example of HTML Comments

This paragraph will be displayed.

In the example above, the comments provide information about the code but do not affect how the page is rendered.
