HTML Link

In HTML, you can create hyperlinks using the <a> element. The href attribute specifies the URL of the page the link goes to.

Example of an HTML Link

Here is an example of a simple hyperlink: Visit Example.com

In this example, clicking on the link will open "https://www.example.com" in a new tab (due to the target="_blank" attribute).

Relative vs Absolute URLs

You can use absolute URLs (full web addresses) or relative URLs (paths relative to the current page) in the href attribute.

  • Absolute URL: https://www.example.com/page.html
  • Relative URL: /page.html

Relative URLs are useful for linking to other pages within the same website, while absolute URLs are necessary when linking to external websites.
