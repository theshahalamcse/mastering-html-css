About HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure and content of a webpage, allowing developers to define elements such as headings, paragraphs, links, images, and more. HTML uses tags to indicate different types of content and their relationships, making it an essential component of web development.

Example of HTML Structure

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Sample HTML Document</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
                <p>This is a sample HTML document.</p>
            </body>
            </html>

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure and content of a webpage, allowing developers to define elements such as headings, paragraphs, links, images, and more. HTML uses tags to indicate different types of content and their relationships, making it an essential component of web development.

  • HTML stands for HyperText Markup Language
  • HTML is the standard markup language for creating web pages
  • HTML uses tags to define elements on a webpage
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Example Explanation

In the example above, we have a basic HTML document structure:

  • <!DOCTYPE html> declares the document type and version of HTML
  • <html> is the root element of the HTML page
  • <head> contains meta information about the document
  • <title> sets the title of the webpage
  • <body> contains the visible content of the webpage
  • <h1> defines a top-level heading
  • <p> defines a paragraph

History of HTML

HTML was created by Tim Berners-Lee in 1990 while working at CERN. The first version of HTML was very simple and had limited features. Over the years, HTML has evolved significantly with the introduction of new elements and attributes to support more complex web pages.

Timeline of HTML Versions

  • 1989: Tim Berners-Lee proposes the concept of HTML.
  • 1991: The first version of HTML (HTML 1.0) is released.
  • 1995: HTML 2.0 is released, introducing new features like tables and forms.
  • 1997: HTML 3.2 is released, adding support for scripting and multimedia.
  • 1999: HTML 4.01 is released, becoming the standard for web development for many years.
  • 2014: HTML5 is finalized, introducing new elements for multimedia, graphics, and improved semantics.
  • 2017: HTML5.1 is published, adding minor updates and improvements to the HTML5 specification.
